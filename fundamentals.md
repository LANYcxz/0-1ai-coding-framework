# 基本面深度分析指南

> **核心原则**：先搞懂这门生意，再看它的财务数据。顺序不能反。

---

## 一、商业模式深度解析

### 1.1 收入结构拆解

**关键问题**：钱从哪里来？
```
产品销售型：一次性，复购依赖产品力
服务订阅型：MRR/ARR驱动，黏性高（偏好此类）
平台佣金型：轻资产，随GMV增长（警惕监管风险）
广告收入型：B端依赖，景气周期敏感
利息/费率型：金融类，注重风险定价能力
```

**实战检验点**：
- 收入集中度：前5大客户占比是多少？超过50%是危险信号
- 合同负债/预收账款：这个数字在增长 = 客户愿意预付 = 信任感强 ✅
- 递延收入：SaaS公司的生命线，持续增长代表未来收入有保障

### 1.2 成本结构分析

```
高毛利（>60%）：软件、品牌消费品、医药
中毛利（30-60%）：消费电子、工业品
低毛利（<30%）：零售、制造、农业
```

**毛利率的核心意义**：
- 毛利率 > 50% + 可持续 → 有定价权，有品牌/技术壁垒
- 毛利率下滑 > 3个百分点/年 → 竞争恶化，护城河在崩塌

### 1.3 单位经济模型（Unit Economics）

特别重要，尤其对于成长期互联网/消费公司：

```
LTV（客户终身价值）= 平均客单价 × 购买频次 × 客户生命周期
CAC（客户获取成本）= 销售费用 / 新增客户数

LTV / CAC > 3 → 健康的商业模式
LTV / CAC < 1 → 本质上是在烧钱补贴，不可持续
```

## 二、护城河深度识别

### 2.1 五种护城河的识别方法

#### 🛡️ 护城河一：品牌溢价（Pricing Power）

**识别特征**：
- 同类产品，消费者愿意多付X%的价格
- 毛利率显著高于行业平均水平
- 广告费率（广告费/营收）低且效果好

**测试方法**：
> 如果这家公司明天把价格提高20%，会损失多少客户？如果损失<10%，说明品牌护城河存在。

**典型案例**：茅台（酱香茅台无可替代）、苹果（iOS生态锁定）、LV（身份符号溢价）

---

#### 🌐 护城河二：网络效应（Network Effect）

**识别特征**：
- 用户增加，每个用户获得的价值也在增加
- 最典型：社交网络、支付平台、交易所

**网络效应强度分级**：
```
强网络效应：微信、Visa（每新增用户让所有用户受益）
弱网络效应：滴滴（同城效应，跨城无效）
伪网络效应：很多公司声称有但实际没有
```

**识别伪网络效应的方法**：
> 如果用户数量减少一半，剩余用户的体验会大幅下降吗？如果不会，就不存在真正的网络效应。

---

#### 🔒 护城河三：转换成本（Switching Cost）

**识别特征**：
- 客户转向竞争对手需要付出高额代价（时间、金钱、学习成本）
- 高续费率（>90%）或高客户留存率

**转换成本的来源**：
```
数据成本：客户历史数据都在这个系统里（Oracle数据库）
学习成本：员工已经培训好用某系统（SAP ERP）
流程嵌入：产品深度嵌入客户业务流程
合同锁定：长期合同+违约金
```

**量化指标**：净收入留存率（NRR/NDR）
```
NRR > 120% → 即使不新增客户，收入也在增长（最强转换成本信号）
NRR > 100% → 存量客户在扩张
NRR < 100% → 客户在流失，转换成本弱
```

---

#### 💰 护城河四：成本优势（Cost Advantage）

**识别特征**：
- 在同等产品/服务下，成本比对手低20%+
- 来源：规模、地理位置、独特资源、更高效的流程

**中国市场特有的成本优势**：
- 人力成本优势（劳动密集型，早期）
- 供应链整合能力（制造业龙头）
- 政府关系与资源获取（有争议但客观存在）

---

#### 📐 护城河五：规模经济（Economies of Scale）

**识别特征**：
- 随着规模扩大，单位成本持续下降
- 行业前3名集中度在提升

**计算方法**：
```
成本弹性 = 成本增速 / 收入增速
成本弹性 < 0.7 → 存在显著规模经济效应
```

### 2.2 管理层质量深度评估

这是最被散户忽视、但最重要的分析维度之一。

#### 管理层质量评估清单

**① 能力维度**
```
□ CEO是创始人还是职业经理人？
  → 创始人CEO平均比职业经理人创造更多长期股东价值（统计结论）
□ 管理层在困难时期（2020疫情、2022熊市）的决策是什么？
  → 困难时期的行动比顺境时的表态更能说明问题
□ 资本分配历史：并购是否创造价值？回购是否在低估时进行？
  → 乱收购是最常见的价值毁灭行为
```

**② 诚信维度**
```
□ 业绩预测的准确度：过去5年，管理层给的Guidance和实际差多少？
  → 频繁miss guidance = 要么能力不足，要么有意误导
□ 持股比例：管理层自己的钱在不在里面？
  → 大比例持股的创始人通常与股东利益一致
□ 薪酬结构：高管薪酬是否与股东回报挂钩？
  → 纯现金奖金激励 < 股权激励（长期绑定）
□ 关联交易透明度：有没有把公司的钱转移到自己关联公司？
```

**③ 愿景与执行**
```
□ 3-5年战略是否清晰且一致？
□ 过去3年战略执行进度如何？
□ 对资本市场的沟通质量：是否坦诚面对坏消息？
  → 只报喜不报忧的管理层是危险信号
```

## 三、竞争分析（波特五力应用）

```
供应商议价能力：供应商能不能随意提价？（影响毛利率）
购买者议价能力：客户能不能压价？（影响定价权）
新进入者威胁：进入门槛高不高？（决定护城河持久性）
替代品威胁：有没有完全不同的解决方案？（颠覆风险）
行业内竞争：现有竞争激烈程度？（影响利润率）
```

**实战应用**：
> 如果五力中有3个以上都在恶化，哪怕当前财务数据好看，也要警惕。好的财务是过去的，五力决定未来的。

## 四、行业渗透率与增长空间测算

### TAM（总可寻址市场）测算方法

**自上而下法**：
```
全球/全国 目标群体人口 × 潜在渗透率 × 人均消费 = TAM
```

**自下而上法**（更准确）：
```
当前最大用户群的平均消费 × 潜在可覆盖用户数 = TAM
```

### 渗透率阶段的投资含义

```
渗透率 <10%  → 早期市场，高不确定性，需要技术/产品判断
渗透率 10-40% → 黄金成长期，龙头收益最大（最佳买入窗口）
渗透率 40-70% → 成熟期，竞争加剧，关注份额而非总量
渗透率 >70%  → 存量竞争，谨慎投资
```

**关键原则**：
> 渗透率处于10-40%区间的行业龙头，是我长期获得超额收益的主要来源。不要在渗透率>70%的行业找成长股。

## 五、宏观与行业周期位置判断

### 宏观环境对不同行业的影响

| 宏观环境 | 受益行业 | 受损行业 |
|---------|---------|---------|
| 利率上行 | 银行、保险 | 高负债成长股 |
| 利率下行 | 科技、消费 | 银行净息差收窄 |
| 通胀上行 | 资源、能源 | 高固定成本制造业 |
| 经济复苏 | 周期、金融、消费 | 防御性行业（相对）|
| 经济衰退 | 医疗、公用事业、必需消费 | 可选消费、工业 |

### 中国特色：政策周期判断

**政策面判断三步法**：
```
Step 1: 看"两会"政府工作报告中的关键词变化
Step 2: 看主管部委近半年的监管动作方向
Step 3: 看国家队/主权基金近期的持仓动向（透过ETF持仓推断）
```

> ⚠️ **终身教训**：2021年教培行业清零式监管，让我彻底确立了一个原则：在中国，**任何受政策高度管控的行业，都需要加倍的安全边际**。政策风险是无法用财务数字对冲的。
