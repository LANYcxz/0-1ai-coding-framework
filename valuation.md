# 估值方法论 + 实战技巧

> **核心原则**：估值不是算出一个精确数字，而是判断当前股价相对内在价值是"贵了"还是"便宜了"。模糊的正确，胜过精确的错误。

---

## 一、估值的本质与基础

### 1.1 股票价值的来源

```
股票的内在价值 = 企业未来所有自由现金流的现值之和

V = FCF1/(1+r) + FCF2/(1+r)² + FCF3/(1+r)³ + ... + 终值/(1+r)^n

这个公式告诉我们：
1. 未来现金流越大 → 价值越高
2. 增长越快 → 价值越高
3. 折现率（风险）越低 → 价值越高
4. 持续时间越长 → 价值越高

所以：好公司 = 大现金流 + 高增长 + 低风险 + 可持续
      = 理应更贵
```

### 1.2 估值的三大方法体系

```
绝对估值：DCF（贴现现金流）→ 计算内在价值
相对估值：PE/PS/PB/EV-EBITDA → 横向比较
历史估值：当前PE vs 历史PE分位 → 纵向比较

三者结合使用，单一方法必然有盲点
```

## 二、主要估值方法详解

### 2.1 PE估值法（最常用）

```
PE = 股价 / EPS（每股收益）
等价于：市值 / 净利润

使用PE估值的前提：公司已经盈利且盈利相对稳定
```

**PE的正确使用方式：**

**①  纵向比较（历史分位）**
```
当前PE处于历史10%分位 → 极度低估（历史上90%的时间比现在贵）
当前PE处于历史50%分位 → 合理
当前PE处于历史90%分位 → 极度高估

注意：历史高PE也可能是合理的，如果增速预期大幅提升
```

**② 横向比较（同行PE比较）**
```
公司PE明显低于行业均值 → 可能低估（或有原因，需深究）
公司PE明显高于行业均值 → 市场给予溢价（需判断是否合理）
```

**③ PEG法（成长股修正版）**
```
PEG = PE / 年化净利润增速（%）

PEG < 0.5 → 成长股中的极度低估
PEG < 1.0 → 相对合理（彼得·林奇经典标准）
PEG > 2.0 → 高估，增速需要兑现才能支撑估值

使用场景：营收/利润高速成长（>20%/年）的成长股
局限：增速预测本身不确定，高增速公司PEG估值容错率低
```

**PE的陷阱：**
```
周期股不能用PE：
  周期景气顶峰 → EPS最高 → PE最低 → 看起来便宜 → 实际上最贵！
  周期底部     → EPS最低或亏损 → PE最高甚至为负 → 看起来最贵 → 实际上最便宜
  结论：周期股用PB + 景气周期位置判断，不用PE

亏损股不能用PE：亏损时PE为负，无意义
高商誉公司要用调整后PE：扣除商誉减值风险
```

---

### 2.2 EV/EBITDA估值法（资本结构中性）

```
EV = 企业价值 = 市值 + 净有息负债（债务 - 现金）
EBITDA = 息税折旧摊销前利润（近似于经营现金流入）

EV/EBITDA = 企业价值 / EBITDA

优势：排除了资本结构（负债多少）的影响，适合跨公司比较
     特别适合：高折旧行业（制造、地产、电信）、重并购行业
```

**EV/EBITDA行业参考（A股/港股）：**
```
消费必需品（白酒等）：20-40x
医疗器械：20-35x
工业制造：8-15x
电信/公用事业：5-10x
银行（不用EBITDA，用PB）
```

---

### 2.3 PS估值法（亏损成长股专用）

```
PS = 市值 / 营业收入

适用场景：
- 公司当前亏损但高速成长（新能源、互联网早期）
- 盈利波动大但营收稳定

局限：不同商业模式的公司，合理PS差异极大
     SaaS（高毛利）的PS自然高于电商（低毛利）

修正版：EV/Revenue（同样考虑负债）

参考标准（与毛利率挂钩）：
毛利率 70%+ 的公司：合理PS = 5-15x
毛利率 40-70% 的公司：合理PS = 2-8x
毛利率 < 40% 的公司：合理PS = 0.5-3x
```

---

### 2.4 PB估值法（资产类/周期股）

```
PB = 市值 / 净资产（股东权益）

PB < 1 的含义：市场认为这家公司的资产实际价值低于账面价值
             （可能是资产质量差，或商誉减值预期，或持续亏损）

PB < 1 不一定便宜！必须结合ROE：

罗杰斯公式：合理PB = ROE / 所需回报率
如果ROE = 8%，所需回报率 = 10%，则合理PB = 0.8x
如果ROE = 15%，所需回报率 = 10%，则合理PB = 1.5x
```

**PB的适用场景：**
```
银行/保险：核心估值指标（ROE稳定、盈利可预测）
周期底部的资源/重工业：PB<1时往往是好买点
高ROE企业：PB偏高是正常的，不代表高估
```

---

### 2.5 DCF模型——终极估值武器

```
DCF的价值：建立你对公司内在价值的独立判断框架
DCF的危险：垃圾假设进，垃圾价值出（GIGO）

三阶段DCF模型：
阶段1（1-5年）：用详细预测的FCF，折现
阶段2（6-10年）：用逐步减速的增速预测
阶段3（10年后）：用永续增长率计算终值

关键参数及对估值的敏感度：
折现率(WACC)±1% → 估值变化约15-20%
永续增长率±0.5% → 估值变化约10-15%
增速预测±5%    → 估值变化约20-30%

使用原则：
1. DCF给出的是一个区间，不是精确点
2. 保守假设得出的价值 = 安全底线
3. 乐观假设得出的价值 = 梦想估值
4. 当前股价在两者之间且偏向保守侧 → 好买点
```

## 三、估值实战技巧

### 3.1 三重验证法（实战最重要的方法）

**任何估值结论都需要三重验证才能有信心：**

```
验证1 - 绝对估值：DCF估算内在价值
验证2 - 相对估值：与同行PE/PB/EV-EBITDA比较
验证3 - 历史估值：当前在历史分位的哪个位置

结论矩阵：
三者都显示低估 → 💎 高确信度买点
两者低估一者中性 → ✅ 可以布局
一者低估两者高估 → ⚠️ 继续等待
三者都显示高估 → 🔴 等待回调
```

### 3.2 行业景气周期与估值的关系

**不同阶段的估值策略：**

```
景气上行初期（低预期 → 验证）：
  PE通常偏低（市场不信任），此时是最佳买点
  特征：分析师普遍下调，但实际数据开始好转

景气上行中期（验证 → 预期上调）：
  PE随预期上调而扩张，"戴维斯双击"
  股价涨幅 = 利润增长 + PE扩张（双重加速）

景气高峰（高预期充分定价）：
  PE处于历史高位，"完美预期"已定价
  任何低于预期 = 股价大跌

景气下行（预期下调）：
  "戴维斯双杀"：利润下降 + PE收缩
  股价跌幅放大

最佳策略：在景气底部+低预期时布局，高峰期减仓
```

### 3.3 安全边际（Margin of Safety）

**巴菲特最核心的投资原则：**

```
安全边际 = (内在价值 - 当前股价) / 内在价值

安全边际 > 30% → 可以考虑买入（给估算误差留空间）
安全边际 > 50% → 极佳买点，可以重仓
安全边际 < 10% → 不够安全，等待更好时机

安全边际的意义：
1. 补偿你估算内在价值时可能犯的错误
2. 保护你免受不可预见的黑天鹅冲击
3. 是长期资产增值的最重要保障
```

### 3.4 择时与估值的结合（实战精髓）

**基本面定方向，估值定价位，情绪面定时机：**

```
最理想的买入组合：
1. 基本面：好公司（护城河强、增速好、财务健康）
2. 估值：低估（至少历史50%分位以下）
3. 情绪面：市场悲观（板块跌幅大、媒体负面）
4. 催化剂：有可见的驱动因素（新产品、政策支持、竞争改善）

四者同时满足 → 极高胜率，可以重仓
三者满足 → 高胜率，正常仓位
两者以下 → 谨慎布局或等待
```

## 四、估值陷阱识别

### 4.1 低PE不等于便宜——价值陷阱

**以下情况，低PE是陷阱：**
```
陷阱1：盈利即将大幅下滑（明年EPS将腰斩，则真实PE是现在的两倍）
陷阱2：护城河消失，竞争恶化（低估值但是永久性价值损失）
陷阱3：周期顶部的低PE（上文已述）
陷阱4：账面利润高但现金流很差（利润不真实）
陷阱5：高负债公司（低PE可能因为高财务风险）

识别方法：
每次发现"低PE好机会"时，先问：
→ 为什么这么便宜？市场比我知道什么吗？
```

### 4.2 高PE不等于贵——成长溢价

**以下情况，高PE是合理的：**
```
合理的高PE：
① 高速成长中（PEG < 1）
② 护城河极强（如茅台、腾讯历史上）
③ 资产轻、FCF高（软件/品牌公司）
④ 行业天花板远未触碰

注意区分：合理高PE vs 泡沫高PE
合理高PE：增速能够支撑 → 持有甚至加仓
泡沫高PE：增速无法支撑 → 等待或减仓
```

### 4.3 对预期的管理

**市场定价的是"预期的变化"，而非绝对水平：**

```
好公司 + 超预期 → 股价大涨（即使当前PE已高）
好公司 + 低于预期 → 股价大跌（哪怕基本面依然良好）
差公司 + 超低预期后回升 → 股价大涨（最不好操作）

实战原则：
买入前要想清楚：未来1-2年内，什么会让预期上调？
              什么会让预期下调？
如果上调预期的概率远大于下调，且当前估值合理 → 好买点
```

## 五、中国A股估值的特殊考量

### 5.1 A股估值特有因素

```
流动性溢价/折价：
A股流动性溢价明显，同质公司A股PE > H股PE（AH溢价）
AH溢价率 > 50% → A股相对H股贵
AH溢价率 < 10% → A股相对H股合理

北向资金与外资定价权：
外资集中持仓的消费/医药蓝筹，估值逻辑更接近全球比较
北向资金净流入持续加速 → 往往是股价重要支撑

政策驱动的估值扭曲：
军工/半导体/新能源：政策支持期PE可以极高（市场定价未来10年）
这类公司：用PS或PS+技术壁垒评估，而非传统PE

国企折价：
央企/国企历史上估值折价（治理折价 + 效率折价）
近年来"央国企重估"逻辑推动，折价在收窄
估值修复逻辑：ROE改善 + 分红提升 → 驱动PB从折价到合理
```

### 5.2 A股散户特性带来的定价机会

```
情绪化定价特征：
A股散户占比高，短期情绪主导；
极度悲观时，好公司被错杀 → 买点
极度乐观时，差公司被追捧 → 卖点或做空机会

利用方法：
监测换手率：极低换手 = 无人关注 = 冷门好股藏着
监测基金重仓：基金重仓的好公司在基金集中赎回时错杀 = 机会

节奏特征：
A股有典型的"一月效应"、"三月布局效应"、年报季效应
利用这些季节性规律作为买入窗口参考（统计意义，非必然）
```
