## 🚦 Step 1：快速定性（5分钟筛选器）

**先问这5个问题，如有3个以上无法回答，暂停深入研究：**

| # | 问题 | 判断标准 |
|---|------|---------|
| 1 | 这家公司靠什么赚钱？能一句话说清楚吗？ | 说不清 → ⚠️ 警惕 |
| 2 | 它的主要竞争对手是谁？为什么客户选它而不是对手？ | 答不出 → ⚠️ 护城河可疑 |
| 3 | 近3年营收和利润趋势如何？是增长还是萎缩？ | 连续萎缩 → 🔴 慎入 |
| 4 | 管理层近1年有没有大规模减持？ | 有 → 🔴 强烈警惕 |
| 5 | 当前估值处于历史什么分位？ | 历史高位 → ⚠️ 需要强论据支撑 |

## 🌐 Step 2：行业与宏观定位

**核心框架：选对赛道，比选对选手更重要。**

### 行业生命周期判断
```
导入期  → 高风险高赔率，需要极强的研究确定性才能重仓
成长期  → 最佳布局窗口，龙头集中度提升阶段
成熟期  → 稳定分红+回购驱动，估值压缩期
衰退期  → 避免！除非有极特殊的重组/转型逻辑
```

### 必须回答的行业问题
- **竞争格局**：CR3（前3名市占率）是多少？是在集中还是分散？
- **渗透率**：现在渗透率多少？天花板还有多远？
- **政策风向**：是政策支持还是监管收紧（近1年有无重大政策变化）？
- **周期属性**：强周期（原油/钢铁）还是弱周期（消费/医疗）？

> 📌 **实战经验**：强周期行业在景气顶峰时PE最低（利润最高），此时买入是陷阱。要在PE最高（景气底部）时考虑买入。

## 🏗️ Step 3：商业模式与护城河

**详细分析见** → [fundamentals.md](fundamentals.md)

### 商业模式快速判断
| 类型 | 特征 | 举例 |
|------|------|------|
| 平台型 | 网络效应，轻资产，高毛利 | 微信、Visa |
| SaaS/订阅 | 高黏性，可预测收入，负流失率最佳 | Salesforce |
| 消费品牌 | 定价权，强复购 | 茅台、可口可乐 |
| 制造业 | 规模效应，成本优势 | 比亚迪 |
| 资源型 | 周期驱动，资产壁垒 | 中石油 |

### 护城河强度评分（1-5分）
```
品牌溢价（Pricing Power）  → 能提价多少%而不流失客户？
网络效应（Network Effect）→ 用户越多价值越大吗？
转换成本（Switching Cost）→ 客户离开的代价是什么？
成本优势（Cost Advantage）→ 比对手便宜多少？可持续吗？
规模经济（Scale Economy） → 规模扩大能持续降本吗？
```

> ⚠️ **护城河总分 < 2 且 PE > 20倍** → 极高风险，价值陷阱概率大

## 📊 Step 4：财务健康度诊断

**详细解读见** → [financials.md](financials.md)

### 核心财务体检表（5大维度）

**① 盈利能力**
```
毛利率趋势：近3年是↑还是↓？（持续↓ = 竞争加剧或成本失控）
净利率：与同行相比高还是低？
ROE ≥ 15%  → 优质
ROIC > WACC → 企业在创造价值（核心标准！）
```

**② 增长质量**
```
营收增速 vs 利润增速：利润增速 > 营收增速 = 规模效应显现 ✅
利润增速 >> 营收增速 = 警惕！可能在压缩费用（不可持续）
```

**③ 现金流质量（最重要！）**
```
经营性现金流 / 净利润 > 0.8 → 利润质量高 ✅
经营性现金流 / 净利润 < 0.5 → 🔴 高度警惕！应收账款堆利润
自由现金流（FCF）= 经营现金流 - 资本支出
FCF连续为正 → 真实的好生意标志
```

**④ 资产负债健康**
```
有息负债/EBITDA < 3x → 安全
有息负债/EBITDA > 5x → 危险
速动比率 > 1      → 短期偿债无忧
```

**⑤ 资本效率**
```
ROIC（投入资本回报率）：衡量每块钱投入能赚多少
ROIC > 10% 且稳定 → 优质护城河的财务体现
ROIC逐年下降      → 护城河在侵蚀，重新评估
```

## 💰 Step 5：估值判断

**详细方法见** → [valuation.md](valuation.md)

### 估值方法速查表
| 公司类型 | 首选估值法 | 参考指标 |
|---------|-----------|---------|
| 成熟盈利型 | PE / EV-EBITDA | 历史分位 + 同行比较 |
| 高速成长型 | PEG / DCF | PEG < 1 = 合理 |
| 亏损高成长 | PS / EV-Revenue | 与增速匹配度 |
| 重资产型 | PB | PB < 1 可能低估（需结合ROE） |
| 周期型 | PB + 景气周期位置 | 底部PB买入 |

### 三重定价法（实战核心）
```
1. 绝对估值（DCF）→ 内在价值锚定
2. 相对估值（同行PE对比）→ 市场定价参考
3. 历史估值分位 → 当前贵/便宜的相对判断

三者均显示低估 → 强买入信号
三者分歧较大   → 深入研究分歧来源
```

> 📌 **黄金买入区**：好公司 + 合理偏低估值 + 市场短期悲观（情绪面低点）

## 🚨 Step 6：风险红旗扫描

**完整风险清单见** → [risk-checklist.md](risk-checklist.md)

### 立即放弃的"一票否决"信号
```
🔴 管理层近6个月大规模减持（>5%持股）
🔴 审计意见非标准无保留意见
🔴 经营现金流连续2年为负而净利润为正
🔴 应收账款增速远超营收增速（财务造假信号）
🔴 核心客户集中度 > 50%（一个大客户走了就完蛋）
🔴 商誉占净资产比例 > 50%（并购雷区）
🔴 行业处于政策重点整治期（如2021年教培、地产）
```

### 需要深入调查的黄色预警
```
🟡 毛利率连续3年下滑
🟡 存货增速 > 营收增速
🟡 关联交易金额异常
🟡 频繁更换审计机构
🟡 创始人/CEO近期密集发表乐观预测（可能在拉高出货）
```

## ⚖️ Step 7：综合裁决 + 操作建议

### 评分卡（总分100分）
| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 行业赛道质量 | 20分 | 成长期优质赛道20，衰退期0 |
| 商业模式+护城河 | 25分 | 护城河强度×5分 |
| 财务健康度 | 25分 | 5项指标各5分 |
| 估值合理性 | 20分 | 低估20，合理10，高估0 |
| 风险清洁度 | 10分 | 无红旗10，每个红旗-5 |

### 操作建议框架
```
总分 ≥ 80 → 💎 高确信度，可考虑重仓（5-10%仓位）
总分 60-79 → ✅ 中等确信度，适合小仓位试探（2-5%仓位）
总分 40-59 → ⚠️ 继续观察，等待更好的切入点
总分 < 40  → 🚫 当前不适合投资，列入观察名单即可
```
